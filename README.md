# DOA-Music-algorithm_circle-array
通过单层圆形麦克风阵列采集音频，实现MUSIC算法的声源定位。



## music_circle array.py 

主程序。文件内的读取音频文件的代码需要自己更改。



## real 音频

实际采集的音频，采集板为16阵元均匀圆形麦克风阵列，半径为0.08m。
